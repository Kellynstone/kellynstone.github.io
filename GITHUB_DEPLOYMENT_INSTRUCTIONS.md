1. Go to **https://github.com/new** on your Redmi phone

3. Repository name: **kellynstone.github.io** (EXACTLY this name)
4. Description: "The Soul of Winter: Russian Winter in Nairobi"
5. Make it **PUBLIC** (important!)
6. Click **"Create repository"**

### Step 2: Upload Files

### Step 3: Enable GitHub Pages
1. Go to repository **Settings**
