# The Soul of Winter: Russian Winter in Nairobi - Deployment Guide

## Permanent Cloud Deployment Instructions

### Quick Start - Recommended Free Hosting Services:

#### Option 1: Netlify (Recommended - Easiest)
1. Go to https://netlify.com
2. Sign up for free account
3. Drag and drop the `deployment_package` folder
4. Your site will be live instantly with a permanent URL

#### Option 2: GitHub Pages
1. Create a GitHub account (free)
2. Create a new repository named `username.github.io`
3. Upload all files from `deployment_package` folder
4. Enable GitHub Pages in repository settings
5. Site will be live at `https://username.github.io`

#### Option 3: Vercel (Free)
1. Go to https://vercel.com
2. Sign up with GitHub account
3. Import the deployment folder
4. Automatic deployment with permanent URL

#### Option 4: Firebase Hosting (Google - Free)
1. Go to https://firebase.google.com
2. Create new project
3. Install Firebase CLI
4. Run: `firebase deploy`
5. Permanent URL provided

## Support & Troubleshooting

**If audio doesn't play:**
- Check browser allows autoplay (some browsers require user interaction first)
- Click the video play button to start both video and audio

**If images don't load:**
- Ensure all files in `upload/` folder are uploaded
- Check file paths in index.html match your hosting structure

**If video doesn't play:**
- Verify `russian_winter_nairobi_final.mp4` is uploaded
- Check browser supports MP4 video format
